# IP: ICMP, IPv6

## ICMP: Internet Control Message Protocol

✔ 사용자 데이터가 아닌 네트워크 상의 증상(데이터 유실, TTL expire 등등)을 나타내는 control message를 운반하는 프로토콜  
✔ ip 패킷의 데이터 부분에는 발생한 증상에 대한 정보를 담는다.  
✔ 네트워크 상태 진단 등등에 이용
